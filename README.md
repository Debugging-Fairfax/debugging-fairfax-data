# Debugging Fairfax Data

This repository contains structured, inspectable civic data used by the **Debugging Fairfax** application.

The data here represents modeled extracts from official Fairfax County public documents, published in a machine-readable format to support transparency, verification, and reuse.

## What this repository contains

- YAML datasets representing Fairfax County civic and financial information
- Data modeled directly from primary source documents
- Stable structures intended for inspection, comparison, and reuse

Current datasets include:

- Fairfax County FY2026 General Fund disbursements

## What this repository does not contain

- Commentary or analysis
- Policy recommendations
- Opinions or conclusions
- The Debugging Fairfax application code

This repository publishes source data only.

## Data sources

All datasets are derived from official Fairfax County public documents.  
Each YAML file includes source references (document, volume, page) in its metadata.

## Usage

The data in this repository is used by the Debugging Fairfax application and may also be used independently for research, analysis, or tooling.

See individual dataset files for structure and scope details.
